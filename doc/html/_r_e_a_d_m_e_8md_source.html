29 #### Parameters
30 1. Background substraction: built-in Vibe, SuBSENSE and LOBSTER; MOG2 from opencv; MOG, GMG and CNT from opencv_contrib
31 2. Foreground segmentation: contours
32 3. Matching: Hungrian algorithm or algorithm based on weighted bipartite graphs
33 4. Tracking: Linear or Unscented Kalman filter for objects center or for object coordinates and size
34 5. Use or not local tracker (LK optical flow) for smooth trajectories
35 6. KCF, MIL, MedianFlow, GOTURN or MOSSE tracking for lost objects and collision resolving
36 7. Haar face detector from OpenCV
37 8. HOG and C4 pedestrian detectors
38 9. SSD detector from OpenCV and models from chuanqi305/MobileNet-SSD
39 10. YOLO and Tiny YOLO detectors from https://pjreddie.com/darknet/yolo/
40 11. Simple Abandoned detector

42 #### Build
43 1. Download project sources
44 2. Install CMake
45 3. Install OpenCV (https://github.com/opencv/opencv) and OpenCV contrib (https://github.com/opencv/opencv_contrib) repositories
46 4. Configure project CmakeLists.txt, set OpenCV_DIR. If opencv_contrib don't installed then set disable options USE_OCV_BGFG, USE_OCV_KCF and USE_OCV_UKF
47 5. Go to the build directory and run make

49 **Usage:**
50 
51  Usage:
52  ./MultitargetTracker <path to movie file> [--example]=<number of example 0..3> [--start_frame]=<start a video from this position> [--end_frame]=<play a video to this position> [--end_delay]=<delay in milliseconds after video ending> [--out]=<name of result video file> [--show_logs]=<show logs> [--gpu]=<use OpenCL>
53  ./MultitargetTracker ../data/atrium.avi -e=1 -o=../data/atrium_motion.avi
54  Press:
55  * 'm' key for change mode: play|pause. When video is paused you can press any key for get next frame.
56  * Press Esc to exit from video
57 
58  Params:
59  1. Movie file, for example ../data/atrium.avi
60  2. [Optional] Number of example: 0 - MouseTracking, 1 - MotionDetector, 2 - FaceDetector, 3 - PedestrianDetector, 4 - MobileNet SSD detector, 5 - YOLO detector
61  -e=0 or --example=1
62  3. [Optional] Frame number to start a video from this position
63  -sf=0 or --start_frame==1500
64  4. [Optional] Play a video to this position (if 0 then played to the end of file)
65  -ef=0 or --end_frame==200
66  5. [Optional] Delay in milliseconds after video ending
67  -ed=0 or --end_delay=1000
68  6. [Optional] Name of result video file
69  -o=out.avi or --out=result.mp4
70  7. [Optional] Show Trackers logs in terminal
71  -sl=1 or --show_logs=0
72  8. [Optional] Use built-in OpenCL
73  -g=1 or --gpu=0
