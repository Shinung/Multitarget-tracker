__GTL_BEGIN_NAMESPACE edge::edge ( )

Default constructor. Creates an invalid edge. The only way to obtain a valid edge is through graph::new_edge. Example: 

  graph g;
  node n1, n2;
  edge e;
  n1 = g.new_node();
  n2 = g.new_node();
  e = g.new_edge(n1, n2);
